На даній сторінці будуть описані базові команди Git!

repositiry (репозиторій)- це місце (папка/директорія) де зберігається інформація про ваш код та його зміни.

commit - фіксує всі зміни які відбулися із репозиторієм.

git init - проводить ініціалізацію проекту в поточній папці (це означає, що команда git init запускає Git в даному репозиторії. Також після ініціалізації, Git створює приховану папку .git із системними файлами).

git status - відображає статус поточного репозиторію у відповідній гілці.

git add - відстежує зміни файлів (команда git add назва файлу додає файл (файли) назва якого (яких) вказана, команда git add . додає всі файли).

git rm --cached назва файлу - перестає відстежувати зміни у файлі назва якого вказане.

git commit -m "назва коміту" - створює commit з повідомленням.

git branch - показує список гілок, та гілку в якій перебуває користувач.

git branch назва гілки - створює нову гілку (назва якої вказана в команді).

git branch -D назва гілки - видаляє гілку (назва якої вказана в команді).

git checkout назва гілки - перемикається на гілку (назва якої вказана в команді)

git checkout -b назва гілки - одночасно створює та перемикається на гілку (назва якої вказана в команді)

cd ім`я папки/директорії - переходить у вказану папку/директорію.

cd .. - переходить на рівень (папку/директорію) вище.

clear - очищує консоль.
